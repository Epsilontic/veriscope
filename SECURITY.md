# Security Policy

## Reporting a vulnerability

If you believe you have found a security vulnerability in Veriscope, please report it
privately by emailing **craig.holm@protonmail.com**. Please include enough detail to
reproduce the issue and any potential impact.

We will acknowledge receipt within 72 hours and provide updates as we investigate.

## Supported versions

Security fixes are applied to the latest released version and the `main` branch.

## Disclosure policy

Please do not disclose security issues publicly until a fix has been released and an
advisory has been published (when applicable).
