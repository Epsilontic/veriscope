# Code of Conduct

Veriscope is committed to providing a welcoming, inclusive, and safe community for everyone.

## Our standards

We expect all participants to:

- Be respectful, considerate, and collaborative.
- Assume good intent and focus on constructive feedback.
- Respect differing viewpoints and experiences.
- Refrain from harassment, discrimination, or exclusionary behavior.

## Unacceptable behavior

Examples of unacceptable behavior include:

- Harassment, personal attacks, or discriminatory language.
- Publishing private information without consent.
- Sustained disruption of discussions or collaboration.

## Reporting

If you experience or witness unacceptable behavior, please report it to
**craig.holm@protonmail.com**. All reports will be handled discreetly.

## Enforcement

The maintainer may take any action deemed appropriate, including warnings or removal from
project spaces, to address violations of this code of conduct.

## Scope

This code of conduct applies to all project spaces, including GitHub issues, pull requests,
community chats, and any other collaboration venues.
